# Simple Fetch Script
`neofetch` and `pfetch` are bloated, so I wrote my own.

# Configuration
To configure `fetch`, you simply edit it.

## Change ASCII logo
Simply change the `ascii()` function to echo something different. Some examples are listed below.

### Ubuntu
```ascii
             .-.
       .-'``(   )
    ,`\ \    `-`.
   /   \ '``-.   `
 .-.  ,       `___:
(   ) :        ___
 `-`  `       ,   :
   \   / ,..-`   ,
    `./ /    .-.`
       `-..-(   )
             `-`
```
